🌍 World Trade Data Explorer
Welcome to World Trade Data Explorer — an interactive Streamlit app that lets you visualize and analyze global trade flows 🌐📈.
Built for data enthusiasts, economists, and curious minds who want to understand how countries exchange goods and services worldwide.

🚀 Features
🌎 Country-level insights — Explore trade balances, top exports/imports, and partner countries.

📊 Interactive visualizations — Charts and maps that respond to your selections in real time.

🔍 Filter and compare — Choose by country, commodity, or year for flexible analysis.

🧠 Data sources — Powered by reliable global trade statistics (e.g., UN Comtrade / World Bank).

💡 Built with Streamlit — Clean interface, fast performance, and easy sharing.

🛠️ Tech Stack
Framework: Streamlit
Language: Python 🐍
Libraries: pandas, plotly, numpy, pycountry
Data: World trade
